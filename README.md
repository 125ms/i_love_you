# 🥀 Protocolo Mystery: Hello Stranger

Este proyecto es una página de presentación interactiva con una estética **Gothic-Cyberpunk**. Fue diseñada para crear una experiencia inmersiva mediante el uso de efectos visuales dinámicos y audio interactivo.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3 (Efecto Glassmorphism) y JavaScript Vanila.
* **Animaciones:** Uso de Keyframes personalizados para el sistema de murciélagos y pétalos.
* **Audio:** Integración de la API de Spotify y disparadores de audio local (`.mp3`).
* **Despliegue:** GitHub Pages.

## 🦇 Características Principales
* **Sistema de Murciélagos y Pétalos:** Generación aleatoria de elementos animados en el DOM mediante JS para mantener la atmósfera oscura.
* **Riff de Guitarra Interactivo:** Implementación de la función `playRiff()` activada por eventos de *mouseover*, rindiendo homenaje al tono cálido de Slash.
* **Contador de Visitas:** Persistencia de datos mediante `localStorage` para rastrear las visualizaciones de la página.
